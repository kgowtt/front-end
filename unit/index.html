mkdir '創建資料夾'

cd '資料夾'

npm init -y 指令會將所在目錄設為npm package，也就是會新增一個package.json檔

npm install --save-dev mocha

npm install --save-dev chai

"scripts": { "test": "mocha || true" }, 在package.json中加入上面脚本，接著就可以透過 npm test 執行 mocha 測試

在根目錄新建test資料夾/test.js文件 => console.log('hello world'); npm test 測試文件需要放在test目錄下， mocka 執行時會執行 test 目錄下的所有 js 文件(僅限 test 以下這層，對於 test/subtest 這層並不執行)

除了在命令行運行，Mocha還可以在瀏覽器運行。
