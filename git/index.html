GITHUB上傳步驟

  • 步驟一:移動到要下載到的位置
    cd '桌面路徑'
  • 步驟二:建立一個專案(Clone with HTTPS)
    git clone '下載git clone位置'
  • 項目更新
  • 步驟三:更新項目
    cd '移動到項目資料夾'
  • git status . git status(縮寫為 git st) 就是確認專案資料夾裡面是否有檔案新增或刪除，紅色文字表示新增的，GitHub沒有這些資料
  • git add . 新增的檔案加入
  • git status 確認剛新增的檔案是否成功加入
  • git commit -m '更新敘述'
  • git push '上傳git clone位置'
  • '打帳密'
